Part 7 - Deploy Your Project with GitHub Page:
Your project should be viewable through a live GitHub Pages link. 

This allows reviewers to open and test your application without downloading your code.

GitHub Pages works well for simple HTML, CSS, and JavaScript projects. GitHub’s documentation says Pages can publish from a repository branch and source folder, such as the root folder / or /docs.
Requirements
Your deployed project should include:
* A working GitHub Pages link
* An index.html file
* CSS and JavaScript files correctly connected
* No private or sensitive information
Basic Deployment Steps
1. Push your finished project to GitHub.
2. Make sure your main HTML file is named:
index.html
1. Open your repository on GitHub. 
2. Click Settings. 
3. Click Pages in the sidebar. 
4. Under Build and deployment, choose: 
    * Source: Deploy from a branch 
    * Branch: main 
    * Folder: /root 
5. Click Save. 
6. Wait a few minutes. 
7. Copy the live GitHub Pages URL and add it to your README. 
GitHub notes that a Pages site needs an entry file like index.html, index.md, or README.md, and for a standard static web project, index.html is the expected choice.

Before You Submit
During your admissions interview, you'll have the opportunity to discuss your project with one of our reviewers.
While every interview is a little different, you should be prepared to answer questions like:
* What problem does your application solve?
* What value does it create for the user?
* Why did you choose this solution?
* How did you decide what to build first?
* How did AI help you during development?
* Was there a time you questioned or challenged an AI suggestion?
* What was the most interesting bug you encountered, and how did you solve it?
* How did you verify that your application worked correctly?
* If you had another weekend to improve your project, what would you add or change?
* What part of your project are you most proud of, and why?
These questions aren't designed to catch you off guard or test your memory. They're meant to help us understand how you think, how you solve problems, and how you work alongside AI.

If you can answer these questions confidently, you've already demonstrated many of the habits we value in this program.
